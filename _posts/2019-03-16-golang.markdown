```go
package main
 
import "fmt"
 
var deckSize int
 
func main() {
  deckSize = 50
  fmt.Println(deckSize)
}
```

function 밖으로는 initialize는 되지만, 값을 할당하는 것만 안된다.

### Type Conversions
```go

func main() {
  greeting := "Hi there!"
  fmt.Println([]byte(greeting))
}
```
